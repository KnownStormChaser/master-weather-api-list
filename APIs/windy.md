### Windy API

Developer API from Windyty, S.E. — the Czech company behind the popular Windy.com — exposing the same data that powers Windy.com across three separate products: a Point Forecast API (raw multi-model data for coordinates), a Map Forecast API (Windy's interactive weather map with models, layers, and isolines), and a Webcams API (the world's largest webcam repository). Each product is priced and keyed separately.

- **Based in:** Prague, Czech Republic (operated by Windyty, S.E., founded 2014; in June 2024 Windyty acquired a majority stake in Switzerland's meteoblue AG, so the two are now corporately linked)
- **Access model:** Instant self-serve — register for an API key per product; the free Testing/Free tiers and the paid Professional tiers are all self-serve key upgrades. Higher request volumes are "by agreement" (contact)
- **Pricing model:** Freemium per product (free Testing/Free tier on each; flat-rate annual Professional tiers; ECMWF and higher volumes are paid/negotiated add-ons)
- **Coverage / data types:** three distinct APIs —
  - **Point Forecast:** machine-readable current forecast for given coordinates (latest run only — no historical/past forecasts); ~20+ parameters (wind, gusts, temperature, dew point, precipitation incl. type, CAPE, cloud layers, humidity, pressure, geopotential height, waves/swell, plus CAMS air-quality fields — CO, dust, SO₂); models: AROME, ICON-EU, GFS, GFS-Wave, NAM (CONUS/Hawaii/Alaska), CAMS
  - **Map Forecast:** embeddable interactive Windy map — 6 weather models, 40+ layers, isolines, with controls/legend/picker; layers span wind, precipitation, snow, clouds, waves/swell, currents, CAPE, air quality (CO/SO₂/NO₂/PM2.5/ozone/aerosol), satellite, fire intensity, and more
  - **Webcams:** access to Windy's global webcam network — listing, metadata, current images, and an embeddable timelapse player
- **Free tier:** per product — **Point Forecast** "Testing": 500 requests/day but returns *randomly shuffled and slightly modified data*, development-only, not for production; **Map Forecast** "Testing": 500 sessions/day, limited models (GFS) and layers (wind/temperature/pressure), development-only; **Webcams** "Free": link/embed only, limited image size, image-URL validity 15 min, listing offset capped at 1,000
- **Paid tiers (annual flat rate):** **Point Forecast** Professional €990/year (10,000 req/day, unlimited models & parameters); **Map Forecast** Professional €990/year (10,000 sessions/day, unlimited models/layers) **+ optional €1,000/year for ECMWF**; **Webcams** Professional €9,990/year (ad-free, full-size images, 24-hour image-URL validity, offset to 10,000, full webcam-list endpoint). Volumes "can be increased by agreement"
- **Rate limits:** per-day quotas — Point Forecast 500/day (Testing) → 10,000/day (Pro); Map Forecast 500/day → 10,000/day (sessions); higher by agreement
- **Output formats:** JSON (Point Forecast data); JS map widget / tiles (Map Forecast); JSON + image/timelapse assets (Webcams)
- **Geographic strength:** global; model selection skews to publicly available NWP (GFS/ICON/NAM/AROME/CAMS), with regional NAM domains for the US, Hawaii, and Alaska
- **Notable limitations:** **ECMWF is excluded from Point Forecast entirely due to licensing**, and in Map Forecast it's an internal-use-only paid add-on (€1,000/year) with certain layers/isolines — thunderstorms, cloud tops/base, visibility, sea temperature, extreme wind/temp/rain, freezing-altitude isoline — available *only* with ECMWF; Point Forecast returns the latest run only (no historical data); the Point Forecast free tier returns deliberately corrupted data (testing only); pricing is flat-rate EUR per product per year (three separate subscriptions), not metered; Webcams Professional is comparatively expensive (€9,990/year)
- **Built on:** publicly available NWP models surfaced directly — GFS & GFS-Wave (NOAA), ICON-EU (DWD), NAM (NOAA, regional US/Hawaii/Alaska), AROME (Météo-France), GEOS-5 (NASA), HRRR (NOAA, map only), and CAMS (Copernicus, air quality); ECMWF is available only as a licensed paid add-on (map) and not at all (point). This is raw model data as shown on Windy.com, not a proprietary blend; for blended/high-resolution or historical data, Windy points users to sister company meteoblue
- **Docs:** https://api.windy.com/ (Point Forecast, Map Forecast, and Webcams each have their own docs)
