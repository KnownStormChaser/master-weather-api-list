### WillyWeather

Feature-rich Australian weather API from WillyWeather — the same service that powers its consumer apps, websites, and widgets — delivering forecasts, real-time observations, tides, swell, wind, UV, warnings, and maps for 17,000+ Australian locations, refined from Bureau of Meteorology data. Priced per request, with the rate scaling by both the services enabled and the number of locations subscribed.

- **Based in:** Bondi Junction, New South Wales, Australia (WillyWeather (AU) Pty Ltd; founded 2010)
- **Access model:** Instant self-serve — register through the API cost calculator, select your services and location count, get a key; the Enterprise API is sales contact ("Contact Us")
- **Pricing model:** Freemium (ongoing free allotment of the first 5,000 requests; metered per-request pricing above it; Enterprise custom) — see limitations re: the free-allotment reset cadence
- **Coverage / data types:** 7-day weather forecasts (temperature, weather precis, rainfall amount & probability), real-time weather-station observations (feels-like, humidity, pressure, wind speed/direction/gusts, dew point, rain today/last hour, with trends), tides (times & heights), swell (height/period/direction, coastal), wind, hourly UV index (with extreme-exposure alerts), sunrise/sunset (incl. first/last light), moon phases, and BoM warnings; plus map layers (radar, satellite), location search (placename, postcode, or coordinates), and region précis long-form text. Location types include suburbs, towns, beaches, rivers, parks, lakes, islands, mountains, campgrounds, and sporting venues
- **Free tier:** first 5,000 requests free (reset cadence not documented on publicly accessible pages — could be recurring or one-time; see limitations)
- **Paid tiers:** metered per-request, quoted in AUD, where the per-request rate is built up from the service groups enabled (Weather, Weather Station, Maps, Warnings; Search and Info are included) **and** the number of locations subscribed. For the full four-service bundle: single location AUD $0.0004965/request (~$0.50 per 1,000); 10 locations AUD $0.000867/request (~$0.87 per 1,000); all 17,000 locations AUD $0.0015705/request (~$1.57 per 1,000). Enterprise: custom / contact for pricing (unlimited requests, no throttling, 99.99% uptime SLA, 24/7 support with a 2-hour response guarantee)
- **Rate limits:** standard keys are rate-limited/throttled; specific published thresholds aren't documented on accessible pages. Each forecast type must be individually enabled on the API key or the request fails. Enterprise removes throttling (unlimited requests)
- **Output formats:** JSON (RESTful)
- **Geographic strength:** Australia — 17,000+ locations fine-tuned to specific coordinates, with the most comprehensive Australian tide and beach/swell coverage. (WillyWeather also runs separate US and UK consumer properties sourced from NOAA, but the documented API here covers Australian locations)
- **Notable limitations:** Australia-only API coverage (single-country); the pricing model is unusual and hard to estimate up front — the per-request rate depends on both which service groups you enable and how many of the 17,000 locations you subscribe to, and it's quoted in AUD; forecast types are gated per key (each must be explicitly enabled). The 5,000-request free allotment's reset period isn't stated on accessible pages, and specific standard-tier rate-limit figures aren't published (the API documentation is a JS-rendered page that blocks automated access, so several fine details can only be confirmed through the live register/docs flow). No historical-archive or air-quality endpoints are documented; data is credited to the Bureau of Meteorology and other sources per WillyWeather's data-disclaimer terms
- **Built on:** primarily Australian Bureau of Meteorology data, refined and interpolated to each location's exact coordinates by WillyWeather; wind is an amalgamation of BoM, NOAA, and ECMWF models vetted against real-time observations, rainfall uses BoM ACCESS models, and tides combine BoM standard-port data with the Australian National Tide Tables and state marine-agency offsets
